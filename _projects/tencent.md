# Robot Dynamic Nonprehensile Manipulation Policy Learning.
In general end-to-end reinforcement learning paradigm for dexterous manipulation, the algorithms are always sample-inefficient and hard to transfer to real world robots. So in this project, I establish a robotc manipulation-imitation learning framework. which combines dynamic system and ES algorithms to learn human-like manipulation skills.
