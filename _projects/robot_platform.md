# Intelligent compliant robot platform based on six-axis force sensor.
The intelligent robot platform aims at improving the compliance of robot to conduct contact-rich
manipulation tasks using tandem six-axis force sensor.
- Designed the structure of the platform and constructed its 3D model by Solid Works,
- Developed the force sensor data visualization and robot control software based Qt which was useful for observing real-time sensor data and controlling the robot on computer,
- Proposed an integrated method which could provide effective and robust compensation for force disturbance and achieve high measurement accuracy for the six-axis force sensor,
- Completed the impedance-based compliant control for the robot.
